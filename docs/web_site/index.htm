Cкачать проект

на данной стадии можно только из SVN, выполнив в консоли команду
svn checkout https://ambernet.svn.sourceforge.net/svnroot/ambernet/trunk ambernet
либо используя какой-нибудь графический клиент Subversion, например TortoiseSVN

Собрать из исходников

Для компиляции проекта под Linux потребуется утилита CMake, для компиляции под Windows - IDE Code::Blocks.
В проекте используются библиотеки SDL, SDL_image, SDL_mixer, SDL_net, FreeType.
В случае компиляции под Linux их необходимо установить из репозиториев вашего дистрибутива либо скачать с оффсайтов.
Для компиляции под Windows дополнительно скачивать ничего не потребуется - всё необходимое лежит в папках bin/binw и projects/CodeBlocks/extlib/.

под Linix

можно при помощи утилиты CMake, выполнив в консоли в корневом директории проекта команды
cmake .
make

под Windows

установив предварительно Code::Blocks запускаем файл проекта AmberSkyNetVR_modules.workspace в папке projects/CodeBlocks, после чего в открывшейся среде разработки выбираем в верхнем меню пункт Build/Build.
