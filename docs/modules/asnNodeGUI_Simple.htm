[ asnNodeGUI_Simple ]

Плагин asnNodeGUI_Simple служит для отображения простых элементов GUI в 3d-пространстве. Пока реализован только узел сцены NodeGUI_TextBox

[ NodeGUI_TextBox ]

Класс предназначен для отрисовки в 3d затекстурированного прямоугольника, заполненного текстом

Переменные NodeGUI_TextBox:

  • Texture - имя файла текстуры, которой будет обтянут прямоугольник
  • ListName - имя текстового списка, строки из которого будут выведены в прямоугольник

    Указатели плагина:

  • World - указатель на менеджер мира класса IWorld.
