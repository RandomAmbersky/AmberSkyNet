[ asnFileSystem ]

Плагин служит для реализации файловой системы. На данный момент файлы можно читать из обычной файловой системы и из архивов формата ZIP.

Реализация FileSystemASN отличается от FileSystem тем, что в менеджере файлов FileSystemASN происходит сначала поиск в папке common, а потом уже - по всем другим папкам. Это полезно в том случае, если существуют общие ресурсы для разных миров...

Плагин реализует источники файлов (имена для стандартной переменной [FileManager]DataReader менеджера файловой системы):

  • FSDisk - возможность чтения из директорий диска
  • FSZIP - возможность чтения из архивов ZIP
