[ asnDeviceGL ]

Плагин служит для отрисовки графики и обработки ввода с клавиатуры и мыши.

Переменные плагина

Переменные плагина на данный момент хранятся не в самом плагине, а читаются из глобальных переменных окружения Engine (GetEnvParam, SetEnvParam).
  • [DeviceManager]WindowName - заголовок окна, в котором будет рисоваться графика.
