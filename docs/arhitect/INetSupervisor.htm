[ INetSupervisor ]

Интерфейс менеджера сети NetSupervisor служит для обмена информацией по сети. На данный момент менеджер реализует только открытие/закрытие сокетов TCP/IP и получение/отправку информации. Конкретные высокоуровневые протколы реализуются плагинами-обработчиками (наследуемыми от интерфейсного класса INetProtocolHandler) принятых пакетов информации, которые разбирают полученный по сети массив информации и отправляют ответ, используя для этого функции cетевого менеджера через интерфейсный класс INetSupervisor.

Переменные

BufferSize - размер приемного буфера сообщения, по умолчанию равен 1024 байт.
