[ INetProtocolHandler ]

Интерфейс обработчика сетевого протокола. Сетевой менеджер ( INetSupervisor ) после получения пакетов данных вызывает функцию HandleMessage у обработчика, отвечающего за порт, по которому была получена информация. Обработчик сетевого протокола анализирует полученную информацию и если надо формирует массив данных для ответа, и вызывает функцию отправки отправки пакетов данных у сетевого менеджера SendMsg, передавая в нее указатель на сформированный пакет данных.

Указатели класса

  • Handler - указатель на функцию-обработчик данных в буфере. Вид функции: 
    void hanlde_HTTP(INetProtocolHandler *handler, void *info);
    Где handler - указатель на обработчик, вызвавший эту функцию,
    info - указатель на структуру, передаваемую обработчиком.
