[ IFileSystem ]

Интерфейс предназначен для реализации файловой системы.

Переменные менеджера файловой системы

Переменные на данный момент хранятся не в самом плагине, а читаются из глобальных переменных окружения Engine (GetEnvParam, SetEnvParam).
  • [FileManager]DataReader - имена классов-источников файлов, перечислены через запятую:
  • [Paths]DataPath - путь к "корневой" папке данных. Поиск данных будет производится только в данной папке (ведь правда, нам не надо чтобы движок мог шарится по всем дискам компьютера, да?)
